``eda-report`` User Guide
=========================

Speed up the `exploratory data analysis`_ and reporting process:

Automatically analyze a dataset, and get:

#. Brief descriptions:

   - Summary statistics.
   - Tests for normality.

#. Revealing visualisations.

   - *Box-plots*, *dist-plots*, *normal-probability-plots*, *scatter-plots* and a *correlation heatmap* for numeric variables.
   - *Bar-plots* for categorical variables.

#. A report in *Word* (.docx) format.
